MetaBridge Protocol is a simplified example of a cross-chain value-locking system. Users can lock assets into the contract, simulating a deposit on one blockchain, and an authorized operator can later release the assets ‚Äî simulating finalization on a destination chain.

## ‚ú® Key Features
- **Asset Locking:** Users can deposit native assets into a secure on-chain escrow vault.
- **Authorized Release:** Only the protocol owner/operator can unlock and release assets.
- **Request Tracking:** Each bridge action is stored with a unique request ID for auditability.
- **Event Logging:** Emitted events make off-chain indexers and dApps easy to integrate.
